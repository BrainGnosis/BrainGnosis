CLOUD DEVELOPER JOB DESCRIPTION
=====================================

POSITION: Cloud Developer
LOCATION: Remote / West Lafayette, IN
EXPERIENCE LEVEL: Junior+ (1-2 years)
ROLE TYPE: Multiple Hats - Startup Environment
COMPENSATION: Competitive + Equity Opportunity

ABOUT THE ROLE:
---------------
We're looking for a passionate Cloud Developer to join our fast-growing AI startup. You'll be responsible for deploying AI features to the cloud and building scalable solutions. This is a hands-on role where you'll wear multiple hats and directly impact how our technology reaches users.

As part of our small, mission-driven team, you'll work closely with founders to build cloud infrastructure and ensure our AI technology is accessible and reliable.

KEY RESPONSIBILITIES:
--------------------
• Deploy and maintain AI models and features on major cloud platforms (AWS, Azure, GCP)
• Build and optimize cloud infrastructure for scalable services
• Implement CI/CD pipelines for rapid deployment
• Design and maintain APIs for cloud-based services
• Monitor system performance and scaling
• Ensure security best practices for cloud deployments
• Collaborate with AI/ML team to deploy models
• Support DevOps practices and infrastructure as code

REQUIRED QUALIFICATIONS:
-----------------------
• 1-2 years of experience with cloud platforms (AWS, Azure, or GCP)
• Proficiency in containerization (Docker, Kubernetes)
• Experience with Infrastructure as Code (Terraform, CloudFormation, or similar)
• Knowledge of CI/CD tools (GitHub Actions, Jenkins, GitLab CI)
• Programming skills in Python, Node.js, or Go
• Understanding of API development and microservices architecture
• Basic knowledge of database management (SQL/NoSQL)
• Familiarity with monitoring and logging tools
• Experience with version control (Git)
• Strong problem-solving skills and attention to detail

PREFERRED QUALIFICATIONS:
------------------------
• Experience with serverless computing (Lambda, Cloud Functions)
• Knowledge of ML model deployment and MLOps practices
• Experience with load balancing and auto-scaling
• Understanding of cloud security and compliance
• Previous startup or fast-paced environment experience
• Knowledge of AI/ML workflows and model serving
• Experience with CDN and edge computing
